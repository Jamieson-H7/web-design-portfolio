HTML (Hypertext Markup Language)
...

HTML is the standard markup language used to create web pages. It consists of various elements and tags that define the structure and content of a web document.

Key points about HTML:

  • Structure: HTML provides the basic structure of a web page, including headings, paragraphs, lists, and links.
  • Tags: HTML uses tags to enclose content and define elements. Common tags include <html>, <head>, <title>, <body>, <h1>, <p>, <a>, etc.
  • Attributes: HTML elements can have attributes that provide additional information. For example, the <img> element may have a src attribute to specify the image source.

Examples of tags and their uses
...

  1. <html>: The root element of an HTML document, containing all other elements.
  2. <head>: Contains meta-information about the document, such as the title, character encoding, and linked stylesheets.
  3. <title>: Specifies the title of the web page, which is displayed in the browser's title bar or tab.
  4. <meta>: Provides metadata about the document, including character encoding, author, and description.
  5. <link>: Links external resources, such as stylesheets, to the HTML document.
  6. <body>: Encloses the main content of the web page, including text, images, and other elements.
  7. <h1> to <h6>: Define headings with varying levels of importance, where <h1> is the highest and <h6> is the lowest.
  8. <p>: Represents a paragraph of text, separating content into readable blocks.
  9. <a>: Creates hyperlinks to other web pages, resources, or email addresses.
  10. <img>: Embeds images in the document, specifying the image source and alternative text.

These tags are fundamental for creating structured and well-formatted web pages in HTML.

CSS (Cascading Style Sheets)
...

CSS is a stylesheet language used to control the presentation and layout of web pages. It defines how HTML elements should be displayed on the screen.

Key points about CSS:
...

  • Selectors: CSS uses selectors to target HTML elements for styling. For example, you can use h1 to select all <h1> elements.
  • Properties: CSS properties determine the visual style of elements. Common properties include color, font-size, margin, padding, and more.
  • Classes and IDs: You can use classes and IDs to apply specific styles to individual elements, enhancing flexibility and reusability.
  • Cascading: CSS rules can cascade, meaning that styles can be inherited, overridden, or combined from various sources like external stylesheets, inline styles, and browser defaults.

Selectors and Properties
...

CSS uses selectors to target HTML elements and apply styles to them. It pairs selectors with properties to define how elements should be styled.

Example:

/* Selector */
h1 {
	/* Property and value */
	color: blue;
}

In this example, the h1 selector targets all <h1> elements and sets their text color property to the blue value.

Classes and IDs
...

You can assign classes and IDs to HTML elements, providing a way to target specific elements for styling.

Example:

/* Class Selector */
.button {
	background-color: green; 
}  
/* ID Selector */ 
#header {
	font-size: 24px; 
}

In this example, the class selector .button styles elements with the "button" class to have a green background, and the ID selector #header styles the element with the "header" ID to have a font size of 24px.

External Stylesheets
...

You can create a separate CSS file, OR use an external CSS file, and link it to your HTML document using the <link> element.

Example:

<head>
	<link rel="stylesheet" type="text/css" href="styles.css"/>
</head>

In this example, the HTML document links to an external CSS file named "styles.css."

Inheritance and Specificity
...

CSS properties can be inherited by child elements from their parent elements. Specificity determines which styles take precedence.

Example:

/* Parent element */
.container { 
	color: red; 
}  
/* Child element */ 
/* This will be applied to <p> elements within .container */ 
.container p { 
	color: blue; 
}

In this example, the text color of <p> elements within an element with the class "container" will be blue, while the text of the container will be red.

Comments
...

You can add comments in CSS using /* */ to provide explanations or notes within your stylesheet.

Example:

/* This is a CSS comment */

CSS allows you to control typography, colors, layout, and more, making it a powerful tool for web design and presentation.

JavaScript
...

JavaScript is a programming language that is primarily used for adding interactivity and functionality to web pages. It can be embedded directly into HTML documents or loaded externally.

Key points about JavaScript:

  • Dynamic Behavior: JavaScript allows you to create interactive elements, validate forms, manipulate the DOM (Document Object Model), and respond to user actions.
  • Variables and Data Types: JavaScript supports variables and different data types like strings, numbers, arrays, and objects.
  • Functions: You can define functions in JavaScript to group code and execute it when needed. Functions are essential for code organization and reuse.
  • Event Handling: JavaScript can listen for events like clicks, input changes, and more, enabling responsive and interactive web applications.
  • Libraries and Frameworks: Many libraries and frameworks, such as jQuery, React, and Angular, build on JavaScript to streamline web development and enhance functionality.

Variables and Data Types
...

JavaScript supports various data types, including numbers, strings, booleans, and objects.

Example:

// Variables
let number = 42; 
let message = "Hello, World!";
let isTrue = true;

console.log(number, message, isTrue); // 42 Hello, World! true

In this example, we declare variables for a number, a string, and a boolean value, all using the keyword "let".

There are three main keywords to declare variables in JavaScript:

  1. let - Variables declared with let are block-scoped, meaning they are limited in scope to the block, statement, or expression in which they are defined. They are mutable, making them useful for values that may change.
  2. var - Variables declared with var are function-scoped or globally scoped. They are mutable and their values may be hoisted to the top of their function or global scope.
  3. const - Variables declared with const are block-scoped like let, but they cannot be reassigned after they are initialized.

Functions
...

Functions in JavaScript allow you to group code into reusable blocks. They can take parameters and return values.

Example:

// Function declaration
function greet(name) {
	return "Hello, " + name + "!";
} 
// Function call
let greeting = greet("Jamieson");

console.log(greeting); // Hello, Jamieson!

In this example, we define a function greet that takes a name parameter and returns a greeting with the entered name.

Conditional Statements
...

JavaScript provides conditional statements like if, else if, and else to make decisions in your code.

Example:

for(var i = -10; i < 40; i+=20){
	let age = i;
	if (age > 0 && age < 18) {
		console.log("You are a minor, age " + age + ".");
	} else if (age >= 18) {
		console.log("You are an adult, age " + age + ".");
	} else {
		console.log("Invalid age, age " + age + ".");
	}
}

/* Output
	Invalid age, age -10.
	You are a minor, age 10.
	You are an adult, age 30.
*/

This code checks the value of the age variable and prints a message based on the condition.

Event Handling
...

JavaScript allows you to respond to events like clicks, mouse movements, and keyboard input on an HTML page, enabling interactivity.

let button = document.getElementById("myButton");
button.addEventListener("click", function () {
	alert("Button clicked!");
});

In this example, a click event listener is added to an HTML button element, showing an alert when the button is clicked.

DOM Manipulation
...

JavaScript can manipulate the DOM to change the content or structure of a web page.

Example:

let element = document.getElementById("myElement");
element.innerHTML = "<p>New content</p>";

This code selects an element by its ID and changes its inner HTML content.

Comments
...

You can add comments in JavaScript using // for single-line comments and /* */ for multi-line comments to document your code.

Example:

// This is a single-line comment

/*
	This is a
	multi-line comment
*/

Example Hello World Page
...

<!-- Defines the document as an HTML one -->
<!DOCTYPE html>
<!-- Starts the HTML page -->
<html>
<!-- Starts the head of the page, which is a container for metadata placed between the <html> tag and the <body> tag. -->
<head>
	<!-- Sets the title of the page, or the label on the tab when its opened -->
    <title>Hello World Page</title>
    <!-- CSS, this will define the styles for the elements within the HTML page -->
    <style>
        /* CSS styles to center text and change its color */
        body {
            text-align: center;
            background-color: #f0f0f0;
        }
        /* Targets any elements with the ID of "hello" and changes the font size and color*/
        #hello {
            font-size: 36px;
            color: #3333cc;
        }
    </style>
</head>
<!-- Starts the body of the page, which contains all the contents of the document -->
<body>
	<!-- Creates a DIV (or division) element and sets its ID to "hello" -->
    <div id="hello">Hello, World!</div>
	<!-- JavaScript, this will dynamically control the page and interactions with it -->
    <script>
        // JavaScript code to change the text dynamically
        // Gets the element with ID of "hello"
        const helloElement = document.getElementById('hello');
        // Sets the element to listen for when there is a click on it
        helloElement.addEventListener('click', function () {
            helloElement.textContent = 'Hello, Human!'; // Change the text when clicked
        });
    </script>
</body>
</html>
